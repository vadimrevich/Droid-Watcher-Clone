DroidWatcher (Android Studio version of Sources)
================================================

Android Studio 3.3.0 версия исходных файлов
-------------------------------------------

Данная версия получена путём импорта оригинальных версий исходных файлов пакета из Eclipce в Android Studio и замены версии Android SDK на 26.

В связи с изменением версии SDK из кода были убраны неподдеживаемые функции, а именно — управление закладками и браузером по-умолчанию.

Внимание!
---------

Данная версия DroidWatcher по прежнему ориентирована на оригинальный файл, который больше не регистрирует новых пользователей и вообще выглядит "умершим". Ближайшая задача автора — переделать приложение так, чтобы оно стало доступно для любого url сайта на любом хостинге. Даже путём снижения безопасности доступа к данным на телефоне!
